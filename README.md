# test_blockchain

Requirement:
`blinker==1.6.2
click==8.1.3
Flask==2.3.1
itsdangerous==2.1.2
Jinja2==3.1.2
MarkupSafe==2.1.2
Werkzeug==2.3.2`

Запустить:
1. Клонирование репозитория.

```git clone https://github.com/xishe/test_blockchain```

2. Скачивание библиотек.

```pip install -r requirements.txt```

3. Запуск программы.

```python server.py```

Web on: http://127.0.0.1:5000/
